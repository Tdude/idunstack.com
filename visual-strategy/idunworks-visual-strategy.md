# Idunworks Visual Content Strategy
*Visual Director: AI Assistant*  
*Created: 2026-02-15*

## Executive Summary

Idunworks is a Swedish web studio that bridges human craft and AI innovation. Our visual identity reflects this duality through clean Scandinavian design principles enhanced by Solarpunk aesthetics, emphasizing sustainability, growth, and technological optimism.

## Visual Identity Core

### Brand Positioning
- **Human + AI Partnership**: Visual harmony between organic and digital elements
- **Scandinavian Minimalism**: Clean lines, generous whitespace, functional beauty
- **Solarpunk Aesthetic**: Nature-tech integration, sustainable futures, golden-green palettes
- **Craft-Oriented**: Visible process, making, building, tangible results

### Color System

#### Primary Palette
- **Idun Gold**: `#D4AF37` - Norse goddess inspiration, premium warmth
- **Forest Green**: `#2D5016` - Swedish nature, growth, sustainability
- **Stone Gray**: `#8B8680` - Nordic rocks, foundational stability
- **Clean White**: `#FAFAFA` - Scandinavian purity, breathing room

#### Secondary Palette  
- **Solar Orange**: `#FF8C42` - Energy, innovation, sunrise
- **Sky Blue**: `#4A90A4` - Swedish skies, clarity, horizon
- **Deep Charcoal**: `#2C2C2C` - Code backgrounds, sophistication
- **Sage Green**: `#9CAF88` - Living systems, organic tech

#### Semantic Colors
- **Success**: `#16A085` - Teal growth
- **Warning**: `#F39C12` - Amber energy  
- **Error**: `#E74C3C` - Controlled red
- **Info**: `#3498DB` - Clear blue

### Typography System

#### Primary: Inter
- Modern, clean, excellent readability
- Tech-forward but human
- Multiple weights available

#### Secondary: JetBrains Mono  
- Code snippets and technical content
- Monospace clarity
- Developer-friendly

#### Display: Playfair Display
- Headlines and hero content
- Touch of elegance
- Norse mythology gravitas

### Logo & Brand Mark Specifications

#### Primary Logo
- Wordmark: "Idun Works" in Inter Medium
- Golden apple symbol (stylized, minimal)
- Horizontal layout for headers
- Vertical stack for smaller spaces

#### Secondary Marks
- "IW" monogram for favicons
- Apple symbol standalone
- Text-only version for constraints

#### Usage Guidelines
- Minimum size: 120px width
- Clear space: 2x logo height
- Never stretch or distort
- Always on white or very light backgrounds for primary version

## Iconography System

### Lucide Icons Implementation
Using Lucide React/Svelte components for consistency:

#### Core Icons
- **Code**: `<Code2 />` - Development work
- **Layers**: `<Layers />` - CMS/stack visualization  
- **Zap**: `<Zap />` - Fast performance
- **Users**: `<Users />` - Team collaboration
- **Leaf**: `<Leaf />` - Sustainability/Solarpunk
- **Globe**: `<Globe />` - Web presence
- **Settings**: `<Settings />` - Technical configuration
- **Sparkles**: `<Sparkles />` - AI enhancement
- **Tree**: `<TreePine />` - Swedish/nature connection
- **Sun**: `<Sun />` - Solar energy/optimism

#### Icon Styling
- Stroke width: 1.5px (balanced visibility)
- Size: 24px standard (16px small, 32px large)
- Color: Inherit from parent or Stone Gray
- Hover: Transition to Idun Gold

### Custom Icons
For unique brand elements not in Lucide:
- Golden apple (Idun reference)
- Human+AI partnership symbol
- Minimal Swedish flag element

## Image Style Guidelines

### Photography Style
#### Team Photos (Tibbe + Buttler Partnership)
- **Environmental portraits**: Modern Scandinavian interiors
- **Natural lighting**: Large windows, soft shadows
- **Casual professional**: Comfortable but competent
- **Tech context**: Laptops, code, modern workspace
- **AI visualization**: Subtle screens, code, collaborative tools

#### Workspace Photography
- **Clean setups**: Minimal desk, quality tools
- **Process shots**: Development in action
- **Technology focus**: Screens with code, modern hardware
- **Swedish elements**: Natural wood, plants, clean lines

#### Client Work Documentation
- **Before/after comparisons**: Performance improvements
- **Device mockups**: Clean, modern presentation
- **Process documentation**: Wireframes to final product
- **Analytics/results**: Data visualization

### Illustration Style
#### Solarpunk Elements
- **Nature-tech integration**: Organic shapes with geometric precision
- **Golden hour lighting**: Warm, optimistic atmosphere
- **Sustainable futures**: Green energy, efficient systems
- **Growth imagery**: Plants, progress, evolution

#### Technical Illustrations
- **System diagrams**: Clean, minimal technical drawings
- **Code visualization**: Elegant syntax highlighting
- **Architecture drawings**: Modern, systematic layouts
- **Data flow**: Clear, directional, purposeful

### Image Specifications

#### Hero Images
- **Dimensions**: 1920x1080px minimum (16:9)
- **Format**: WebP primary, JPG fallback
- **Quality**: 85% compression
- **Content**: Wide, atmospheric, story-driven

#### Feature Illustrations  
- **Dimensions**: 800x600px (4:3) or 1200x800px (3:2)
- **Format**: SVG when possible, WebP for complex images
- **Style**: Consistent with brand colors and Solarpunk aesthetic

#### Screenshot Galleries
- **Dimensions**: 1200x900px (4:3) for desktop, 375x812px (9:19.5) for mobile
- **Format**: PNG for UI precision, WebP for web delivery
- **Annotations**: Subtle callouts using brand colors
- **Presentation**: Clean device frames, consistent lighting

#### Profile Images
- **Dimensions**: 400x400px square
- **Format**: JPG for photos, PNG for graphics
- **Style**: Professional but approachable
- **Background**: Consistent with brand palette
